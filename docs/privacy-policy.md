# Politique de confidentialité — Ça joue !

*Dernière mise à jour : 7 mars 2026*

## 1. Présentation

**Ça joue !** est une application mobile gratuite d'apprentissage des expressions suisses romandes. Elle est développée à titre personnel et n'est affiliée à aucune entreprise.

## 2. Données collectées

L'application utilise **Firebase** (Google) pour collecter de manière anonyme :

- **Rapports de plantage** : informations techniques en cas d'erreur (type d'appareil, version du système d'exploitation, trace de l'erreur).
- **Statistiques d'utilisation** : données anonymes telles que le nombre de sessions, la durée d'utilisation et les écrans consultés.

Ces données ne permettent pas de vous identifier personnellement.

## 3. Données NON collectées

L'application **ne collecte pas** :

- Votre nom, adresse e-mail ou numéro de téléphone
- Votre localisation
- Vos contacts ou photos
- Aucune donnée personnelle identifiable

Aucun compte utilisateur n'est requis pour utiliser l'application.

## 4. Finalité

Les données collectées servent uniquement à :

- Détecter et corriger les bugs
- Comprendre l'utilisation générale de l'application pour l'améliorer

Aucune donnée n'est vendue, partagée à des fins publicitaires ou transmise à des tiers, à l'exception de Google Firebase qui traite les données en tant que sous-traitant.

## 5. Services tiers

L'application utilise les services suivants de Google Firebase :

- **Firebase Crashlytics** — rapports de plantage
- **Firebase Analytics** — statistiques d'utilisation anonymes

Vous pouvez consulter la politique de confidentialité de Google à l'adresse suivante : [https://policies.google.com/privacy](https://policies.google.com/privacy)

## 6. Stockage et conservation

Les données sont stockées sur les serveurs de Google (Firebase). Les données d'analyse sont conservées pendant **14 mois**, conformément aux paramètres par défaut de Firebase Analytics.

## 7. Mineurs

L'application est adaptée à tous les âges. Elle ne collecte aucune donnée personnelle, y compris celle de mineurs.
